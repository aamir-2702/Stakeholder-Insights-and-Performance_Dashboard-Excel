##  **📌 Overview**

This project analyzes stakeholder performance and financial metrics using interactive dashboards and KPIs to support data-driven decision-making.

---

## **🎯 Objective**

- Analyze premium trends over time
- Evaluate insurer performance and value creation
- Understand profitability across cities and tenure
- Provide data-driven insights for business decision-making

## **Key Metrics**

- KPI Tracking: Total Premium, Avg Stakeholder Age, Equity Value
- Interactive filters: City, Stakeholder, Gender, Date, Tenure
- Visualizations:
      - Premium Trends Over Time
      - Value by Insurer
      - Profitability by City & Tenure
      - Shareholding Distribution
      - Cost Ratio vs Profitability
- Drill-down, tooltips, and cross-filtering
- PDF Export functionalit

---

## **📌 Key Insights**
- Bancassurance channels generate the highest value
- Long-term policies show better profitability
- Certain cities outperform others in profitability
- Identified high-performing insurers contributing maximum value
- Observed trends in premium growth over time
- Analyzed profitability variations across cities and tenure
- Highlighted cost vs profitability relationship
  
---

## **💡 Recommendations**

- Optimize cost ratio to improve profitability
- Focus on high-performing regions and insurers
- Improve strategies for low-performing segments
- Use data-driven pricing and operational strategies
